# 🍽️ Smart Restaurant Recommender - Quick Reference

## ⚡ Start in 3 Steps

### Windows
```bash
cd "Smart Restaurant Recommender"
run.bat
```

### Linux/Mac
```bash
cd "Smart Restaurant Recommender"
bash run.sh
```

### Manual
```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📱 Using the App

1. **Select Cuisines**: Choose from dropdown (Japanese, Italian, etc.)
2. **Set Budget**: Drag sliders to set budget range
3. **View Results**: See restaurants sorted by rating
4. **Export**: Download as CSV if needed

## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| "Module not found" | `pip install -r requirements.txt` |
| "Dataset not found" | Check `data/restaurants.csv` exists |
| "Port in use" | `streamlit run app.py --server.port 8502` |
| Slow app | Restart or reload browser |
