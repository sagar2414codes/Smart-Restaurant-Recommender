🎯 RECOMMENDATION ALGORITHM
════════════════════════════════════════════════════════════════════════════

Step 1: CUISINE FILTERING
   → Check each restaurant's cuisines
   → Match selected cuisines (case-insensitive)
   → Include if ANY cuisine matches

Step 2: BUDGET FILTERING
   → Check average cost for two
   → Include if: min_budget ≤ cost ≤ max_budget

Step 3: RATING FILTERING
   → If min_rating > 0: exclude unrated restaurants
   → Include if: rating ≥ min_rating

Step 4: RANKING
   → Primary: Sort by rating (descending)
   → Secondary: Sort by votes (descending)
   → Tertiary: Sort by name (ascending)

Step 5: RESULTS
   → Return top N recommendations
   → Display with full details
   → Allow CSV export


✨ HOW TO USE THE APP
════════════════════════════════════════════════════════════════════════════

1. OPEN THE APP
   → Browser opens to http://localhost:8501

2. LEFT SIDEBAR - SET YOUR PREFERENCES
   → 🍜 Select Cuisines: Choose 1 or more (Japanese, Italian, etc.)
   → 💰 Budget Range: Drag sliders to set min-max budget
   → ⭐ Minimum Rating: Optional rating filter (0-5)
   → 📊 Show Results: Choose how many restaurants to see

3. MAIN AREA - VIEW RECOMMENDATIONS
   → See restaurants sorted by rating (best first)
   → Each restaurant shows:
     • Name and location
     • Star rating & review count
     • Cost for two people
     • Cuisines offered
     • Address and features

4. EXPORT RESULTS
   → Click "Export Results as CSV"
   → Download recommendations to your computer
